2808. 使循环数组所有元素相等的最少秒数

给你一个下标从 0 开始长度为 n 的数组 nums 。

每一秒，你可以对数组执行以下操作：

对于范围在 [0, n - 1] 内的每一个下标 i ，将 nums[i] 替换成 nums[i] ，nums[(i - 1 + n) % n] 或者 nums[(i + 1) % n] 三者之一。
注意，所有元素会被同时替换。

请你返回将数组 nums 中所有元素变成相等元素所需要的 最少 秒数。

示例 1：

输入：nums = [1,2,1,2]

输出：1

解释：我们可以在 1 秒内将数组变成相等元素：

  • 第 1 秒，将每个位置的元素分别变为 [nums[3],nums[1],nums[3],nums[3]] 。变化后，nums = [2,2,2,2] 。
    1 秒是将数组变成相等元素所需要的最少秒数。

示例 2：

输入：nums = [2,1,3,3,2]

输出：2

解释：我们可以在 2 秒内将数组变成相等元素：

  • 第 1 秒，将每个位置的元素分别变为 [nums[0],nums[2],nums[2],nums[2],nums[3]] 。变化后，nums = [2,3,3,3,3] 。
  • 第 2 秒，将每个位置的元素分别变为 [nums[1],nums[1],nums[2],nums[3],nums[4]] 。变化后，nums = [3,3,3,3,3] 。
    2 秒是将数组变成相等元素所需要的最少秒数。

示例 3：

输入：nums = [5,5,5,5]

输出：0

解释：不需要执行任何操作，因为一开始数组中的元素已经全部相等。

提示：

1 <= n == nums.length <= 10^5

1 <= nums[i] <= 10^9

Solution

加起来折腾了三个多小时，有点羞耻。

一开始的想法是，次数最少肯定要全部数字变成出现最频繁的数（记为mark）。所以就往这个方向写了。

最初的方法是check每轮中，nums[i]前后的数字是否为mark，并且不能在当轮check完马上改变数值，所以还加了visitor数组去实现。

然后问题来了，最频繁多的数有多个，这样要如何选呢？没有判别标准，所以将数数功能外置成一个函数（这样就可以保留原来的nums数组）。折腾了一阵也实现了。

然后出现了一个例子[1, 11, 11, 11, 19, 12, 8, 7, 19]。告诉我从一开始的想法就不对。

于是观察了这个数组的形状，解法应该是检查相同数相邻的距离。即以每一个数作为原点，像水波一样往左右扩散，直至碰到与自身相同的数。那么最少的轮次，（选定某一个数后）就是这个数与相同数相邻隔的最远距离除以二（最远相邻才可以保证全部数都被覆盖到）。

这个方法是需要检查所有数并记录它们出现的位置，使用unordered_map辅助记录。

class Solution {
public:
    int minimumSeconds(vector<int>& nums) {
        unordered_map<int,vector<int>> hash;
        int mark=0,n=nums.size();
        int ans=INT_MAX;
        for(int i=0;i<n;i++){
            hash[nums[i]].push_back(i);
        }
        for(auto pos:hash){
            int m = pos.second[0] + n - pos.second.back();
            for(int j = 1;j<pos.second.size();j++){
                m = max(m,pos.second[j]-pos.second[j-1]);
            }
            ans = min(ans,m/2);
        }
        /*for(auto i = hash.begin();i!=hash.end();i++){
            int m = i->second[0] + n - i->second.back();
            int temp = 0;
            for(int j=1;j<i->second.size();j++){
                temp = max(m,(i->second[j]-i->second[j-1]));
            }
            ans = min(ans,temp/2);
        }*/
        return ans;
    }
};
